# Summary of SAT Changes in Shasta v1.3

Shasta v1.3 included version 2.2.3 of the `sat` python package and CLI.

This version of the `sat` CLI contained the following commands:

- `auth`
- `bootsys`
- `cablecheck`
- `diag`
- `firmware`
- `hwinv`
- `hwmatch`
- `k8s`
- `linkhealth`
- `sensors`
- `setrev`
- `showrev`
- `status`
- `swap`
- `switch`

See the [System Admin Toolkit Command Overview](#system-admin-toolkit-command-overview)
and the table of commands in the [SAT Authentication](#sat-authentication) section
of this document for more details on each of these commands.
