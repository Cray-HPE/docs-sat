# SAT Changes in Shasta v1.3

Shasta v1.3 included version 2.2.3 of the `sat` python package and CLI.

This version of the `sat` CLI contained the following commands:

- `auth`
- `bootsys`
- `cablecheck`
- `diag`
- `firmware`
- `hwinv`
- `hwmatch`
- `k8s`
- `linkhealth`
- `sensors`
- `setrev`
- `showrev`
- `status`
- `swap`
- `switch`

For more information on each of these commands, see the
[SAT Command Overview](../about_sat/introduction.md#sat-command-overview) and the table
of commands in the [SAT Command Authentication](../about_sat/command_authentication.md)
section of this document.
